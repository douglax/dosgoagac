Elementos del sistema

El sistema cuenta con un conjunto de componentes gráficos estándar que permiten una interacción fácil con el usuario. El sistema está basada en una tecnología multiplataforma por lo que la representación final de los elementos en las diferentes pantallas dependerá del sistema operativo en el cual se ejecute el programa.
Para todos los ejemplos e ilustraciones contenidos en esta ayuda se emplearon los componentes de Microsoft Windows, sin embargo, los usuarios de MacOs o linux no tendrían ningun problema en identificar los elementos propios de su ambiente.
Cuando se instala ecofactura y es ejecutado por primera vez, se presenta la pantalla de la siguiente ilustración la cual nos muestra la interface en su forma mas elemental.
